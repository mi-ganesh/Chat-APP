## How Run This:

# create a .env and inside it
- PORT=3000
- MONGO_URI=your_mongodb_connection_string
- JWT_SECRET=your_secure_secret
- NODE_ENV=development

Backend Setup:
- cd Backend
- npm install
- npm run dev

FrontEnd setup:
- cd Frontend
- npm install
- npm run dev
